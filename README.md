## 🚀 About

> **Fine-tuned ResNet-50 achieving 91% Top-5 accuracy on CUB-200 dataset**

**Image Classifier App** is a production-ready **AI/ML** project built with **Python, PyTorch, Gradio, AWS S3**.
Deployed on **HuggingFace Spaces** with full CI/CD pipeline.

## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/alam025/bird-classifier.git
cd bird-classifier

# 2. Install
pip install -r requirements.txt

# 3. Run
python main.py
```
